# 프로젝트 기획

## 1. 프로젝트 개요

- **주제:** 프롬프트 저장 및 공유
- **목적:** 바이브코딩(Vibe Coding) 시 AI 성능을 극대화하는 사용자 지침(Custom Instructions) 및 효율적인 프롬프트를 공유하고 탐색하는 공간 제공
- **대상:** 바이브코딩 유저, AI 툴 헤비 유저, 각 분야별 AI 활용 크리에이터

---

## 2. 사이트맵 및 화면 배치

### 메뉴 구성 (Sitemap)

- **Main:** 프롬프트 검색 (구글 검색 처럼 정중앙 검색창), ai 질문창
- **Explore:** 카테고리별 탐색 (바이브코딩, 글쓰기, 이미지 생성 등)
- **Submit:** 나만의 핵심 프롬프트 공유 및 등록
- **Library:** 유저가 북마크(저장)한 프롬프트 보관함

- 로고: "Prompt Archive"
- 메뉴 내비게이션: Main, Explore, Submit, Library (아이콘 포함)
- 주제별 카테고리 섹션: 바이브코딩, 글쓰기, 이미지 생성, 기타
1. Central Search Area (우측 상단 영역):
    - 검색창: 중앙 정렬, 돋보기 아이콘
    - 검색창 밑에 ai 대화창
2. Trend Cards Grid (우측 하단 영역):
    - 카드 구성 요소: 프롬프트 제목, 작성자, 좋아요(추천) 수, 태그 리스트, 본문 요약문

위 스펙을 만족하는 컴포넌트 코드를 작성해줘. 완성도 높은 UI 컴포넌트 형태로 코드를 제공해줘.

[Role]
너는 플랫폼의 핵심 기능인 '프롬프트 상세 보기 페이지'를 설계하는 UI/UX 전문가야.

[Objective]
사용자가 특정 프롬프트 카드를 클릭했을 때 진입하는 상세 페이지를 구현해줘. 메인 페이지에서 작성한 좌측 사이드바 레이아웃은 그대로 유지되어야 해.

[UI Layout Specification]

1. Header Area:
    - 프롬프트 제목 (예: "Cursor AI 전용 바이브코딩 최적화 시스템 지침")
    - 등록일
    - 우측 액션 버튼 인터페이스: 좋아요(추천) 버튼, 보관함 저장 버튼
2. Prompt Content Block (핵심 영역):
    - 사용자가 복사해 갈 프롬프트 본문이 노출되는 박스. 배경색을 일반 배경보다 어둡게 처리하여 코드 블록 형태로 가독성을 높여줘.
    - 우측 상단에 고정된 'Copy' 버튼 배치 (클릭 시 클립보드 복사 기능 활성화 및 텍스트가 'Copied!'로 변경되는 인터랙션 포함).
3. Usage Tips Section:
    - "💡 이 프롬프트는 이렇게 사용하세요!" 섹션 추가. 작성자가 권장하는 AI 모델(Claude 3.5 Sonnet, GPT-4o 등) 및 사용 팁을 보여주는 영역.

위 가이드라인에 맞춘 상세 페이지 컴포넌트 코드를 작성해줘.
